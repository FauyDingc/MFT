# 代码改进总结

## 新增功能

### 1. fd-item 命令 - 创建物品
- **支持物品类型**：2D物品和3D物品
- **参数说明**：
  - `name`: 物品名称
  - `des`: 物品描述
  - `type`: 物品类型（"2d" 或 "3d"）
  - `texture`: 纹理/模型名称
  - `edible`: 是否可食用（可选，默认为false）
  - `effect`: 食用效果（可选）
  - `hand`: 手持效果（可选，如"sword"）

### 2. fd-block 命令增强
- **新增参数**：
  - `explosion`: 爆炸抗性（可选，默认为1.0）
  - `light`: 发光等级（可选，0-15，默认为0）

## 使用示例

```bash
# 创建模组
fd-mod {name:"我的模组",des:"一个测试模组"}

# 创建带特殊属性的方块
fd-block {name:"发光石头",des:"会发光的石头",icon:"glow_stone",explosion:"3.0",light:"10"}

# 创建2D物品（钻石剑）
fd-item {name:"钻石剑",des:"锋利的钻石剑",type:"2d",texture:"diamond_sword",hand:"sword"}

# 创建可食用的3D物品（苹果）
fd-item {name:"魔法苹果",des:"恢复生命的苹果",type:"3d",texture:"magic_apple",edible:"true",effect:"regeneration"}
```

## 文件结构

```
模组目录/
├── manifest.json          # 模组清单
├── scripts/
│   └── mod.js            # 基础脚本
├── blocks/               # 方块定义
├── items/                # 物品定义
├── textures/
│   ├── blocks/          # 方块纹理
│   └── items/           # 物品纹理
├── models/
│   ├── block/           # 方块模型
│   └── items/           # 物品模型（3D物品）
└── texts/
    └── zh_CN.lang       # 语言文件
```

## 注意事项

1. 所有纹理文件需要手动放置到相应目录
2. 3D物品需要额外的模型文件
3. 参数值需要用双引号包围
4. 先创建模组，再添加方块或物品
5. 可选参数可以省略，使用默认值

## 兼容性

- 支持 Minecraft Bedrock Edition 1.16.0+
- 生成的JSON文件符合Minecraft附加包标准
- 支持中文语言文件
